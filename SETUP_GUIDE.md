# SETUP GUIDE — Deploy RegSearch

## No API key or server needed — completely free!

---

## STEP 1 — Push to GitHub

1. Go to https://github.com and create a new repository called `regsearch`
2. Make it **Public**
3. Upload these files:
   - `index.html`
   - `README.md`
   - `.gitignore`

## STEP 2 — Enable GitHub Pages

1. Go to **Settings → Pages**
2. Source: Deploy from a branch
3. Branch: `main`, folder: `/ (root)`
4. Click **Save**

Your site will be live at:
`https://YOUR-GITHUB-USERNAME.github.io/regsearch/`

---

## How it works

- **Cached search** — instant results from 45+ built-in documents, appears as you type
- **Live search buttons** — after searching, click any agency button to open their official search page with your query pre-filled in a new tab
- **No API key, no server, no cost**
